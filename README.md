# 🎓 Student Graduation Prediction

## 📌 Problem
Memprediksi apakah siswa akan lulus berdasarkan jam belajar dan kehadiran.

## 📊 Dataset
Dataset sederhana dengan fitur:
- Jam belajar
- Kehadiran

## ⚙️ Model
- Random Forest Classifier

## 📈 Hasil
- Accuracy: 0.85 (contoh)

## 🚀 Tech Stack
- Python
- Scikit-learn
- Pandas
- Matplotlib
